## 🚀 How to Use

### View the Application
**URL**: http://localhost:5174

### Test as Student
```
1. Click "Student" role button
2. Enter any email (e.g., student@test.com)
3. Enter any password
4. Click "Sign In"
5. View dashboard, submit feedback, see results
```

### Test as Admin
```
1. Click "Admin" role button
2. Enter any email (e.g., admin@test.com)
3. Enter any password
4. Click "Sign In"
5. Create forms, view feedback, see analytics
```

### Terminal Commands
